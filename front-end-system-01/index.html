# 技术栈

# 前端

  • 语言： typeScript

  • 前端框架： vue 3

  • 脚手架： vite

  • 路由： vue-router 4

  • 状态管理： pinia

  • CSS 预编译处理： sass

  • 网络请求工具： axios

  • 前端 UI 框架： element-plus 2

# 后端

  • 语言： javaScript
  • 运行环境： node.js 14
  • 后端开发框架： express 4
  • 数据库： mysql
  • 缓存数据库： redis
  • 数据库映射模块（ORM）： sequelize 5

# 前端项目构建

  • node 版本：16.17.0
  • npm 版本：8.19.2
