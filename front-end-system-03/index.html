# 前言

上文我们完成了添加用户的接口，本文我们继续来完成登录接口。要完成登录接口，首先还要三个步骤。

  1. 前端获取登录验证码接口（新的接口）
  2. 验证登录用户信息（用户名、帐户、密码、uuid（验证验证码唯一字段））
  3. 判断用户名是否存在、账号是否停用

uuid 为前端传来的验证图片验证码的唯一字段。前端获取验证码时用 uuid 获取验证码后端返回验证码并将 uuid 及验证码通过 key，value 的形式存在 redis 数据库，然后再将此 uuid 传给登录接口就可进行验证码的验证
